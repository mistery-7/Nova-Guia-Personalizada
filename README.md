# Nova Guia personalizada

Personalize a nova guia de navegadores baseados em Chromium (Google Chrome, Edge, Brave, Vivaldi, etc.)

Uso intuitivo. Basta inserrir a URL desejada para a nova guia e confirmar.

### Como instalar a extensão em navegadores Chromium — instruções rápidas

1. Baixe o código em arquivo .zip, em seguida descompacte-o.
2. Abra o navegador Chromium `(Chrome, Edge, Brave, etc.).`
3. Vá para a página de extensões:
  - Chrome/Brave: `chrome://extensions/`
  - Edge: `edge://extensions/`
4. Ative Modo de desenvolvedor (geralmente no canto superior direito).
5. Clique em Carregar sem compactação (Load unpacked).
6. Selecione a pasta da extensão (a pasta descompactada contendo o manifest.json).
7. Verifique se a extensão aparece na lista e está ativada.
